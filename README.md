# SISTEMA DE GESTION DE WORKFLOW
Desarrollo y documentación del sistema de gestión de workflow



## Integrantes
- David Portilla

## Objetivo
Desarrollar un sistema que permita a supervisores auditar documentos que crean usuarios con el rol de autores y editores.

##  Tecnologías
- Java (backend)
- SpringBoot (Framework)
- SQL (base de datos)
- HTML + Bootstrap (frontend)
- GitHub (control de versiones y gestión del proyecto)
